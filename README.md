# Simple Interest Calculator

This project is a Bash script that calculates simple interest.

## Formula
SI = (P × R × T) / 100

## Inputs
- Principal
- Rate of Interest
- Time Period

## Usage
Run the script:
bash simple-interest.sh

Enter values when prompted.

## Example
Input:
P = 1000
R = 5
T = 2

Output:
Simple Interest = 100
